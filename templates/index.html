What is CARPY?


CARPY is a tool for testing enrichment of protein properties. Currently 40+ properties compiled from SGD and high-throughput analyses are tracked for over 6000 proteins. The database can be queried by gene name, SGID or yeast ORF identifiers

Start a query


  • One ID per line
  • Search by gene name, systematic name (yORF), or SGID
Experimental Group
(Optional) Control Group

How does it work?


The input for CARPY is a list of protein IDs or gene names

CARPY performs statistical analysis of all properties to find those that are significantly enriched or de-enriched in your dataset compared to the proteome More information on the statistical analysis can be found on the about page
